# ふー ふー ふー ふー はー

呼吸 × ひらがな × 円と泡。ブラウザで動くアンビエントBGM作品集。
音はすべて Web Audio API によるリアルタイム生成（著作権フリー）。

YouTube チャンネル「ふー ふー ふー ふー はー」の各動画に対応するアプリを、ここから配布しています。

## 作品

| # | 作品 | 呼吸法 | URL |
|---|------|--------|-----|
| 01 | コヒーレント呼吸（海辺） | 5.5秒吸って5.5秒吐く・約6回/分 | `01-coherent/` |

## 使いかた

各作品ページを開いて「▶ はじめる」を押すだけ。円がふくらむと吸う、しぼむと吐く。

- 音はページを開いている間だけ、その場で生成されます
- スマホでは画面を点けたままにしてください（消灯すると音が止まります）

## 技術

- 単一HTMLファイル / 依存なし
- Web Audio API（パッド・ドローン・泡・汽笛はすべて合成音）
- Canvas 2D（呼吸円・ペースリング・泡・水平線の灯り）
