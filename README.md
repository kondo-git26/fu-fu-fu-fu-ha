# ふー ふー ふー ふー はー

呼吸 × ひらがな × 円と泡。ブラウザで動くアンビエントBGM作品集。
音はすべて Web Audio API によるリアルタイム生成（著作権フリー）。

作品の基本指標は **BrPM（Breaths Per Minute／1分あたりの呼吸数）**。
YouTube チャンネル「ふー ふー ふー ふー はー」の各動画に対応するアプリを、ここから配布しています。

## 作品

| # | 作品 | 呼吸法 | BrPM | URL |
|---|------|--------|------|-----|
| 01 | コヒーレント呼吸 | 5.5秒吸って5.5秒吐く | 約6 | `01-coherent/` |

## 使いかた

各作品ページを開いて「▶ はじめる」を押すだけ。円がふくらむと吸う、しぼむと吐く。

- 起動前から、ひらがな（すう↔はく）と円は静かに呼吸しています
- 画面に触れると右上に情景（背景色：濃紺・墨・鉄色）を選ぶスウォッチが現れます。呼吸・音はそのままに、まとう色だけが替わります
- 音はページを開いている間だけ、その場で生成されます
- スマホでは画面を点けたままにしてください（消灯すると音が止まります）

## 設計思想

「曲」ではなく「呼吸する音の環境」を設計する。BrPM に関わる骨格（呼吸・倍音・ドローン・汽笛）は作者が握り、背景色のようにBrPMに影響しない要素だけを鑑賞者に委ねる。

## 技術

- 単一HTMLファイル / 依存なし
- Web Audio API（パッド・ドローン・泡・汽笛はすべて合成音。汽笛は純正倍音で和音に共鳴)
- Canvas 2D（呼吸円・ペースリング・泡・背景）
